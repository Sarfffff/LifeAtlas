# 岁迹 | LifeAtlas

岁迹是一款基于地图、时间轴、照片和文字的人生记录 App。

它的目标不是做普通相册或旅行打卡工具，而是把用户走过的地方、拍过的照片、写下的文字和重要人生节点，沉淀成一张可回看的“人生地图”。

## 一句话定位

把走过的地方、拍过的照片、写下的文字，变成一张可回看的生命地图。

## 当前阶段

当前处于 V0.1 本地记录闭环开发阶段。第一版优先验证最小闭环：

```text
新增文字记录 -> 本地保存 -> 时间轴展示 -> 详情页查看
```

目前已完成项目框架、Room 数据库接入、首页概览统计、首页最近记录列表、新增记录写入、发生日期选择、照片导入、编辑记录照片管理、照片私有缩略图缓存、照片失效占位提示、时间轴首图展示、时间轴卡片摘要优化、简单标签、标签筛选、标签空状态提示、标签管理、时间轴搜索、搜索结果数量提示、JSON 数据导出与导入、JSON 导入前预览、地点坐标录入、地图点位列表、地图选点入口占位、地图 SDK 选型、地图 SDK 接入准备、地图配置状态展示、高德地图 SDK 依赖接入、地图页真实地图展示、地图记录 marker 展示、地图 marker 点击查看详情、地图选点页面、地图选点地址反查、当前位置选择、设置页本地优先开关持久化、记录详情查看、编辑和删除。完整照片备份能力会在后续模块中逐步接入。
