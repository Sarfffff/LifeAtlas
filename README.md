# 岁迹 | LifeAtlas

岁迹是一款基于地图、时间轴、照片和文字的人生记录 App。

它的目标不是做普通相册或旅行打卡工具，而是把用户走过的地方、拍过的照片、写下的文字和重要人生节点，沉淀成一张可回看的“人生地图”。

## 一句话定位

把走过的地方、拍过的照片、写下的文字，变成一张可回看的生命地图。

## 当前阶段

当前处于 V1.0 本地体验版。第一版已经完成本地优先生命地图的核心闭环：

```text
新增记录 -> 本地保存 -> 时间轴/地图展示 -> 详情查看与编辑 -> 备份导出与恢复
```

目前已完成项目框架、Room 数据库接入、首页概览统计、记录新增/详情/编辑/删除、发生日期选择、照片导入与私有缩略图缓存、时间轴首图展示、标签管理与筛选、时间轴搜索、JSON 数据导出与导入、完整备份包导出与导入预览、备份包媒体恢复、地点坐标录入、地图真实展示、记录 marker、地图选点、地址反查、当前位置选择、设置页本地优先开关、版本信息、发布准备清单、隐私政策草案和 V1.0 回归测试清单。后续进入真机回归、应用图标和正式签名准备。
