# V1.0 体验说明

本文档用于安装和体验岁迹 V1.0 正式体验版，并集中记录后续改进意见。

## 1. 当前版本

```text
versionCode = 11
versionName = 1.0.0
```

## 2. 可安装 APK

优先体验 Debug 包：

```text
E:\岁迹\app\build\outputs\apk\debug\app-debug.apk
```

Release 未签名包位置：

```text
E:\岁迹\app\build\outputs\apk\release\app-release-unsigned.apk
```

说明：
- Debug 包适合当前自己安装体验。
- Release 包已完成混淆和资源裁剪构建，但正式分发前仍需要 release keystore 签名。

## 3. 建议体验顺序

1. 首次启动 App，确认首页、底部导航和设置页可正常打开。
2. 新增一条文字记录，填写标题、正文、日期、地点、心情和重要程度。
3. 为记录添加照片，保存后检查时间轴首图和详情页照片。
4. 新增或编辑标签，在时间轴按标签筛选。
5. 使用时间轴搜索标题、正文、地点或心情。
6. 打开地图页，检查未配置或已配置高德 Key 时的提示和展示。
7. 新增或编辑记录时使用地图选点。
8. 设置页导出 JSON 备份，并查看导入预览。
9. 设置页导出完整备份包，并查看导入预览。
10. 编辑记录并删除记录，检查数据是否同步刷新。

## 5. 当前已知边界

- 暂未接入登录和云同步。
- 地图功能依赖高德 Key；未配置 Key 时只显示配置提示。
- 完整备份包只恢复 App 私有媒体缓存，不恢复系统相册中的原始照片。
- 正式发布前仍需要 release keystore、高德控制台发布签名 SHA1、隐私政策正式版审阅和真机回归记录。
