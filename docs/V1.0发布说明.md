# V1.0 发布说明

版本：`1.0.0-rc1`

日期：2026-06-24

## 版本定位

V1.0 候选版聚焦个人使用场景，优先保证本地记录、时间轴回看、地图定位、照片关联和数据备份恢复可以形成完整闭环。当前版本适合作为正式发布前的真机回归版本。

## 主要功能

- 本地优先记录：支持标题、正文、发生日期、地点、经纬度、心情和重要程度。
- 时间轴回看：支持按时间展示记录、搜索标题/正文/地点/心情，并结合标签筛选。
- 标签体系：支持创建标签、记录关联标签、标签筛选和标签管理入口。
- 照片能力：支持通过 Android Photo Picker 导入照片，并生成 App 私有缩略图缓存。
- 地图能力：接入高德地图 SDK，支持地图页真实地图、记录 marker、marker 点击进入详情、地图选点、地址反查和当前位置选择。
- 详情与编辑：支持记录详情查看、编辑保存、照片管理和删除确认。
- 数据备份：支持 JSON 导出/导入、导入前预览、完整备份包导出、完整备份包导入预览和媒体缓存恢复。
- 设置页：支持本地优先开关持久化、地图配置状态、版本信息和发布前基础信息查看。

## 技术状态

- Android 原生应用，Kotlin + Jetpack Compose。
- Room 作为本地数据库，已保留 schema 并补充迁移测试。
- DataStore 保存轻量设置。
- Coil 负责图片加载。
- Kotlin Serialization 负责结构化备份数据。
- 高德地图 SDK 用于地图展示和点位交互。

## 发布前已准备

- 版本号已更新为 `versionCode = 10`、`versionName = 1.0.0-rc1`。
- Release 构建支持从本机 `local.properties` 读取签名配置。
- keystore 和签名文件已加入 Git 忽略规则。
- 已补充发布准备清单、隐私政策草案和 V1.0 回归测试清单。

## 已知边界

- 暂未接入账号体系和云同步。
- 暂不恢复系统相册中的原图，只恢复备份包内包含的 App 私有媒体缓存。
- 地图功能依赖高德 Key；未配置 Key 时应展示配置提示。
- 正式发布前仍需完成真机回归、正式应用图标、release keystore 和应用市场隐私合规检查。

## 建议回归重点

- 新增、编辑、删除记录。
- 照片导入、照片显示和编辑移除。
- 标签创建、标签关联和筛选。
- 地图 Key 未配置和已配置两种状态。
- 地图选点、地址反查和定位权限拒绝后的手动选点。
- JSON 备份导出/导入。
- 完整备份包导出/导入和媒体缓存恢复。
