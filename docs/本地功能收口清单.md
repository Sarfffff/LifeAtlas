# 本地功能收口清单

记录日期：2026-06-26

本清单只记录不依赖阿里云邮件推送、域名备案或云端同步的功能。

## 已完成

- 首页：照片轮播、统计卡、写一段新记忆、最近记忆、长按删除。
- 新增记录：标题、正文、时间、地点、坐标、心情、重要程度、标签、照片导入。
- 编辑记录：修改基础信息、替换照片、重新地图选点。
- 详情页：查看记录、照片区、收藏、系统分享、编辑、删除确认和删除失败提示。
- 时间轴：搜索、分类筛选、收藏筛选、卡片直接收藏/取消收藏、重要程度星级展示。
- 地图页：真实地图背景、城市筛选、搜索、点亮新城市、收藏、分享、城市详情、温馨点亮页切换。
- 城市详情：按城市聚合记录、进入记录详情、直接系统分享。
- 收藏中心：集中查看收藏、取消收藏、直接系统分享。
- 设置页：头像昵称签名修改、记录偏好、标签管理、地图配置说明、隐私权限说明。
- 数据背包：完整备份包导出、JSON 导出、备份包/JSON 导入预览和恢复。
- 本地账号：本地登录状态、跳过登录、本地账号安全说明、忘记密码界面骨架。

## 仍需外部服务

- 真实邮箱验证码发送：等待阿里云邮件推送、发信域名和服务器配置稳定。
- 邮箱验证码注册闭环：需要后端真实发信可用后再做端到端测试。
- 云同步：需要账号体系稳定后，再实现云端数据表、媒体上传、冲突合并和软删除同步。
- 发布上线：需要备案、隐私政策正式版、Release 签名和应用市场材料。

- 城市详情新增系统分享入口。
