# 备案后上线验证清单

记录日期：2026-07-04

## 1. 更新服务器后端

在阿里云服务器执行：

```bash
cd /opt/lifeatlas/LifeAtlas/server
bash deploy/update-lifeatlas-auth.sh
```

如果脚本没有执行权限，可以先执行：

```bash
chmod +x deploy/update-lifeatlas-auth.sh deploy/check-lifeatlas-auth.sh
```

## 2. 验证健康检查

期望返回包含 `mailConfigured`：

```bash
curl https://api.lifeatlas.cn/health
```

示例：

```json
{"ok":true,"service":"LifeAtlas Auth Server","mailConfigured":true}
```

含义：

- `ok:true`：后端服务在线。
- `mailConfigured:true`：服务器 `.env` 中 SMTP 字段完整。
- `mailConfigured:false`：SMTP 字段缺失或仍是占位值。

## 3. 验证邮件链路

App 内测试顺序：

1. 注册页输入 QQ 邮箱。
2. 点击获取注册验证码。
3. 检查 QQ 邮箱收件箱和垃圾箱。
4. 用验证码完成注册。
5. 退出登录。
6. 使用邮箱验证码登录。
7. 进入忘记密码，发送重置验证码并完成重置。

## 4. 常用排查命令

```bash
cd /opt/lifeatlas/LifeAtlas/server
bash deploy/check-lifeatlas-auth.sh
```

查看实时日志：

```bash
sudo journalctl -u lifeatlas-auth -f
```

编辑服务器配置：

```bash
sudo nano /opt/lifeatlas/LifeAtlas/server/.env
sudo systemctl restart lifeatlas-auth
```

## 5. SMTP 必填配置

```env
SMTP_HOST=smtpdm.aliyun.com
SMTP_PORT=465
SMTP_SECURE=true
SMTP_USER=no-reply@mail.lifeatlas.cn
SMTP_PASS=阿里云邮件推送SMTP密码
SMTP_FROM="岁迹 <no-reply@mail.lifeatlas.cn>"
```

## 6. 通过标准

- `https://api.lifeatlas.cn/health` 返回 200。
- `mailConfigured:true`。
- QQ 邮箱能够收到注册验证码。
- 邮箱验证码登录可用。
- 忘记密码验证码可用。
- App 不再出现 Firebase 超时或连接重置类错误。
