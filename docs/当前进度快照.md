## 当前阶段

项目处于 V1.0 正式体验版阶段。

V1.0 的目标是完成一个不依赖登录、不依赖云端、以 Room 本地数据库为核心的个人可用版本：

```text
新增记录 -> 本地保存 -> 时间轴查看 -> 详情查看 -> 编辑/删除 -> 备份导入导出
```

1. 启动 App。
2. 首页查看统计数据。

- 地图核心链路已经基本打通。

- 删除记录时清理对应缓存文件。

## 本轮新增进度：完整备份包媒体恢复

当前已接入 zip 备份包恢复：

- zip 预览后可以确认导入。
- 导入会恢复 Room 中的记录、照片引用、标签和标签关联。
- 导入会把备份包内媒体缓存写回 App 私有目录。
- 照片表会重写 `thumbnailPath` 和 `compressedPath`。
- 覆盖旧数据后会清理旧照片缓存文件。
- 如果数据库导入失败，会清理本次已恢复的媒体文件。

当前边界：

- 只恢复备份包内的 App 私有媒体缓存。
- 不主动复制或恢复旧设备系统相册原图。
